HTML Tables

In this article we will go over what HTML tables are and how we put one together. This will provide you indepth knowledge and samples of code to help you visualize how to construct one.

So what are HTML tables? HTML tables are just like a spread sheet that you use with Microsoft Excel with the exception that they are used on a webpage and don't do the mathmatical calculations by default, you have to program that in. Tables allow web developers to create rows and columns that help arrange data into a view format.

Creating the header

Making an HTML table is pretty simple! You start with a simple <table> and close with </table>. Just like a spread sheet you will need to have a header, column, and rows. Lets start with the header and see how to assemble one. When you create a header in a HTML table you start with <thead> and close with </thead>. Now you add a row in-between the header which involoves <tr> and ends with </tr>. Now that we have that, we'll add the table header columns and to do that we open with <th> and close with </th>.

Creating the body

To create the body we follow the same set of instuction as before with making the header but there are some syntax differences. Westart with <tbody> and close with </tbody>. Now you add a row in-between the header which involoves <tr> and ends with </tr>. Now that we have that, we'll add the table data columns and to do that we open with <td> and close with </td>.

Creating the footer

To create the table footer we follow the same set of instuction as did with the body and header but we have one syntax difference now. Westart with <tfoot> and close with </tfoot>. Now you add a row in-between the header which involoves <tr> and ends with </tr>. Now that we have that, we'll add the table data columns and to do that we open with <td> and close with </td>.

Here is an example of what the code should look like

<table>
 <thead>
  <tr>
   <th>Column 1</th>
   <th>Column 2</th>
   <th>Column 3</th>
  </tr>
 </thead>
 <tbody>
  <tr>
   <td>Data column 1</td>
   <td>Data column 2</td>
   <td>Data column 3</td>
  </tr>
 </tbody>
 <tfoot>
  <tr>
   <td colspan="2">Footer column 1 and 2</td>
   <td>Footer column 3</td>
  </tr>
 </tfoot>
</table>
